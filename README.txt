Bulender - README

How to run locally:
1. Place the project folder in a web server root (XAMPP: htdocs, Laragon: www).
2. Make sure PHP is enabled to use contact.php (for sending emails).
3. Open in browser: http://localhost/bulender_project/index.html

Files:
- index.html      (main homepage)
- css/style.css   (styles)
- js/script.js    (interactions)
- contact.php     (server-side handler for contact form)
- images/         (placeholder images for demo)
- README.txt
- LICENSE.txt

To prepare for sale:
- Replace logo and texts with buyer's company name.
- Change the email in contact.php ($to variable) to buyer's email.
- Zip the folder and provide demo link on hosting if needed.
